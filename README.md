## Dependencies

- neovim nightly (0.5+)
- paq-nvim - A plugin manager, it will handle everything else. [here](https://github.com/savq/paq-nvim.)
- Nodejs, for coc.nvim to work.

## Installation

```bash
git clone https://github.com/MoltenCoreDev/ssovim ~/.config/nvim
```
